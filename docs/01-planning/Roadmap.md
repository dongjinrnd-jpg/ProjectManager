| 주차 | 작업 내용 | 산출물 |

| 7주 | 업무일지 CRUD (담당자 자동입력) | 작성, 수정, 삭제, 전체 열람 |
| 8주 | 업무일지 ↔ 프로젝트 자동 연동 | 자동 반영 |
