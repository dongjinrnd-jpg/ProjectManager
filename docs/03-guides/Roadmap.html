1.2 Phase 1: MVP (핵심 기능)

기간: 2월 1주 ~ 4월 1주 (9주)

주차 작업 내용 산출물
1주 프로젝트 세팅, Next.js 초기화 프로젝트 구조
2주 자체 로그인 시스템 (ID/PW) 구현 로그인/로그아웃
3주 사용자 관리 CRUD (관리자) 사용자 생성/수정/삭제
4주 Google Service Account 연동 데이터 읽기/쓰기
5주 프로젝트 목록 페이지 목록 조회, 필터
6주 프로젝트 CRUD (팀장/팀원 구조) 상세 조회, 수정
7주 업무일지 CRUD (담당자 자동입력) 작성, 수정, 삭제, 전체 열람
8주 업무일지 ↔ 프로젝트 자동 연동 자동 반영
9주 테스트 및 버그 수정 MVP 완료

Phase 1 마일스톤:

  • 사용자 관리 CRUD (관리자 전용)

1.3 Phase 2: 대시보드 & 일정

기간: 4월 2주 ~ 6월 1주 (8주)

주차 작업 내용 산출물
1주 대시보드 레이아웃 메인 화면 구조
2주 단계별/소속별 현황 차트 통계 카드

4주 즐겨찾기 기능 중요 프로젝트 모니터링
5주 전체 간트 차트 구현 대일정 차트
6주 개별 일정표 (세부추진항목) 계획/실적 입력
7주 계획 vs 실적 비교 시각화 색상 구분 차트
8주 테스트 및 버그 수정 Phase 2 완료

  • 즐겨찾기 기능 (중요 프로젝트만 보기)
