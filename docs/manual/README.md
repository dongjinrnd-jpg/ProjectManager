## 빠른 시작 가이드

### 1. 로그인
1. 시스템 URL 접속
2. 이메일과 비밀번호 입력
3. **[로그인]** 클릭

### 2. 대시보드 확인
- 로그인 후 자동으로 대시보드 이동
- 본인 권한에 맞는 정보 표시

### 3. 메뉴 이용
- 좌측 사이드바에서 메뉴 선택
- 본인 역할에 따라 표시되는 메뉴가 다름

## 공통 기능

### 날짜 입력 단축키
- `20260211` 입력 → 자동으로 `2026-02-11` 변환
- 모든 날짜 입력 필드에서 동일하게 적용

### 즐겨찾기
- 프로젝트 카드의 별(☆) 아이콘 클릭
- 즐겨찾기한 프로젝트는 대시보드에 우선 표시

### 필터/검색
- 대부분의 목록 화면에서 필터 제공
- 고급 검색 메뉴에서 복합 조건 검색 가능
