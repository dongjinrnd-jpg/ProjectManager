## 1. 시스템 개요

### 1.1 시스템 소개

연구개발 프로젝트 관리 시스템은 프로젝트의 진행 현황을 추적하고 관리하는 웹 기반 시스템입니다.

### 1.2 접근 가능 메뉴

일반 사용자는 다음 메뉴에 접근할 수 있습니다:

| 메뉴 | 아이콘 | 설명 | 권한 |
|------|--------|------|------|
| 대시보드 | 📊 | 전체 프로젝트 현황 요약 | 조회 |
| 프로젝트 | 📁 | 프로젝트 목록 및 상세 | 조회 |
| 전체 일정 | 📅 | 캘린더 형태의 일정 | 조회 |

### 1.3 화면 구성

```
┌─────────────────────────────────────────────────────────────────┐
│                        Header (상단 바)                          │
│  ┌─────┐  프로젝트 관리 시스템              [알림] [사용자 정보] │
│  │Logo │                                                        │
└──┴─────┴────────────────────────────────────────────────────────┘
│                                                                  │
│  ┌──────────┐  ┌────────────────────────────────────────────┐   │
│  │          │  │                                            │   │
│  │ Sidebar  │  │              Main Content                  │   │
│  │ (메뉴)   │  │              (본문 영역)                   │   │
│  │          │  │                                            │   │
│  │ - 대시보드│  │                                            │   │
```

## 2. 로그인

### 2.1 로그인 화면

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│                    ┌─────────────────────┐                      │
│                    │                     │                      │
│                    │    🏭 R&D Project   │                      │
│                    │    Management       │                      │
│                    │                     │                      │
│                    │  아이디             │                      │
│                    │  ┌───────────────┐  │                      │
│                    │  │               │  │                      │
│                    │  └───────────────┘  │                      │
│                    │                     │                      │
│                    │  비밀번호           │                      │
│                    │  ┌───────────────┐  │                      │
│                    │  │ ●●●●●●●●      │  │                      │
│                    │  └───────────────┘  │                      │
│                    │                     │                      │
│                    │  ┌───────────────┐  │                      │
│                    │  │    로그인     │  │                      │
│                    │  └───────────────┘  │                      │
│                    └─────────────────────┘                      │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 로그인 방법

1. 시스템 URL에 접속합니다
2. **아이디**: 관리자에게 부여받은 이메일 주소 입력
3. **비밀번호**: 관리자에게 부여받은 비밀번호 입력
4. **[로그인]** 버튼 클릭

### 2.3 로그인 실패 시

- 아이디 또는 비밀번호가 틀린 경우 오류 메시지가 표시됩니다
- 비밀번호를 잊은 경우 관리자에게 재설정을 요청하세요

### 3.2 주요 기능

#### 현황 카드

| 카드 | 설명 |
|------|------|
| **전체** | 등록된 전체 프로젝트 수 |
| **진행중** | 현재 진행 중인 프로젝트 수 |
| **보류** | 일시 중단된 프로젝트 수 |
| **즐겨찾기** | 즐겨찾기로 등록한 프로젝트 수 |

#### 단계별 현황

각 개발 단계별 프로젝트 수를 보여줍니다:
- 검토, 설계, 개발, PROTO, 신뢰성, P1, P2, 승인, 양산

#### 최근 업무일지 / 이슈 현황

- 최근 작성된 업무일지 목록
- 미해결 이슈 현황

### 3.3 필터 기능

- **[⭐ 즐겨찾기만]**: 즐겨찾기한 프로젝트만 표시
- **[새로고침]**: 최신 데이터로 갱신

### 4.4 탭별 기능

| 탭 | 설명 |

| **💰 원가** | 투자비/원가 정보 (선택 사항) |

### 4.5 즐겨찾기 등록/해제

- 프로젝트 상세 화면의 **[⭐]** 버튼 클릭
- 프로젝트 목록의 별(☆/★) 아이콘 클릭
- 즐겨찾기한 프로젝트는 대시보드에 우선 표시됩니다

## 6. 자주 묻는 질문

### Q1. 업무일지 메뉴가 보이지 않습니다.

> 일반 사용자(user) 등급에서는 업무일지 기능을 사용할 수 없습니다.
> 업무일지 작성이 필요한 경우 관리자에게 권한 변경을 요청하세요.

### Q2. 프로젝트를 수정하고 싶습니다.

> 일반 사용자는 조회만 가능합니다.
> 수정이 필요하면 해당 프로젝트 담당자 또는 관리자에게 요청해주세요.

### Q3. 비밀번호를 변경하고 싶습니다.

> 현재 비밀번호 변경 기능은 관리자만 가능합니다.
> 관리자에게 비밀번호 재설정을 요청해주세요.

### Q4. 즐겨찾기는 왜 사용하나요?

> 즐겨찾기를 등록하면:
> - 대시보드에서 해당 프로젝트를 먼저 볼 수 있습니다
> - [즐겨찾기만] 필터로 관심 프로젝트만 조회할 수 있습니다

### Q5. 로그아웃은 어떻게 하나요?

> 화면 우측 상단의 사용자 정보 영역을 클릭하면 로그아웃 버튼이 표시됩니다.
