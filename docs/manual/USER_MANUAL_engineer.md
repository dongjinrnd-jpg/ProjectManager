#### 상태 자동 계산 로직

| 상태 | 조건 | 색상 |
|------|------|------|
| 예정 | 계획 시작일 > 오늘 AND 실적 시작일 없음 | ⚪ 회색 |
| 진행중 | 실적 시작일 있음 AND 실적 종료일 없음 | 🟠 주황 |
| 완료 | 실적 종료일 있음 | 🟢 초록 |
| 지연 | 계획 시작일 ≤ 오늘 AND 실적 시작일 없음 | 🔴 빨강 |

- **☑️ 이슈사항**: 이슈 내용에서 검색
- 둘 다 체크하면 두 필드 모두에서 검색

1. 검색 조건 설정

4. 우측 상단 **[📂 저장된 검색]** 드롭다운에서 불러오기

### 5.5 결과 정렬

테이블 헤더 클릭으로 정렬 변경:
- 날짜, 프로젝트, 고객사, 단계, 담당자 기준 정렬 가능

## 8. 팁 & 단축키

### 8.1 효율적인 업무일지 작성

1. **매일 작성**: 업무 종료 전 당일 업무 기록
2. **구체적 기록**: "회의 참석" → "A사 킥오프 회의 참석 (요구사항 검토)"
3. **이슈 즉시 기록**: 발생 즉시 기록하여 누락 방지
4. **세부추진항목 연동**: 항목 선택 시 자동으로 실적 반영

### 8.2 날짜 입력 팁

- `20260211` 입력 → 자동으로 `2026-02-11` 변환
- 모든 날짜 입력 필드에서 동일하게 적용

### 8.3 프로젝트 진행 관리

- 단계 변경 시 확인 팝업 표시 → 프로젝트 상태 자동 반영
- 세부추진항목의 **[완료]** 버튼 클릭 → 실적 종료일 자동 입력

## 9. 자주 묻는 질문

### Q1. 다른 사람의 업무일지를 수정할 수 있나요?

> 아니요. 본인이 작성한 업무일지만 수정 가능합니다.

### Q2. 지난 날짜의 업무일지를 작성할 수 있나요?

> 네. 날짜를 선택하여 과거 날짜의 업무일지도 작성 가능합니다.

### Q3. 프로젝트 삭제는 어떻게 하나요?

> 프로젝트 삭제는 관리자만 가능합니다. 관리자에게 요청해주세요.

### Q4. 단계를 변경하면 어떻게 되나요?

> 팀장만 단계를 변경할 수 있습니다. 변경 시:
> - 프로젝트의 현재 단계가 자동 업데이트됩니다
> - 변경 이력이 기록됩니다

### Q5. 검색 조건을 저장하면 어디서 볼 수 있나요?

> 고급 검색 화면 우측 상단의 **[📂 저장된 검색]** 드롭다운에서 확인할 수 있습니다.
