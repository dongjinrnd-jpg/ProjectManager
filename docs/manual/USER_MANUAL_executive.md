## 1. 시스템 개요

### 1.1 경영진 역할

경영진은 프로젝트 전체 현황을 모니터링하고 의사결정을 지원받는 역할입니다.

### 1.2 접근 가능 메뉴

| 메뉴 | 아이콘 | 설명 | 권한 |
|------|--------|------|------|
| 대시보드 | 📊 | 전체 프로젝트 현황 요약 | 조회 |
| 프로젝트 | 📁 | 프로젝트 목록 조회 | 조회 |
| 업무일지 | 📝 | 팀 업무일지 조회 | 조회 |
| 고급 검색 | 🔍 | 업무일지 통합 검색 | 조회 |
| 전체 일정 | 📅 | 캘린더 형태의 일정 | 조회 |
| 주간보고 | 📋 | 주간 단위 업무 보고서 | 조회 |
| 경영진 | 👔 | 경영진 전용 대시보드 | 조회 |

### 2.3 상태 카드 상세

| 상태 | 색상 | 의미 | 조건 |
|------|------|------|------|
| ● 정상 | 파란색/녹색 | 계획대로 진행 중 | 계획 대비 실적 차이 ≤ 10% |
| ▲ 지연 | 노란색/주황색 | 일정 지연 발생 | 계획 대비 실적 차이 > 10% |
| ✓ 완료 | 녹색 | 프로젝트 완료 | 상태가 "완료" |

### 2.4 프로젝트 카드 정보

| 항목 | 설명 |
|------|------|
| **고객사** | 고객사명 |
| **ITEM** | 프로젝트명/코드 |
| **단계** | 현재 진행 단계 (검토, 설계, 개발...) |
| **상태** | 정상/지연 표시 |
| **진행률** | 프로그레스 바 + 퍼센트 |
| **기간** | 프로젝트 시작일 ~ 종료일 |
| **[💬]** | 코멘트 아이콘 (숫자는 미읽은 코멘트 수) |

### 2.5 즐겨찾기 관리

1. 우측 상단 **[⭐ 즐겨찾기 관리]** 클릭
2. 모니터링할 프로젝트 선택/해제
3. 선택된 프로젝트가 대시보드에 카드로 표시

> **권장**: 5~10개 정도의 핵심 프로젝트만 즐겨찾기 등록

## 4. 프로젝트 현황 조회

### 4.1 프로젝트 목록

사이드바에서 **"📁 프로젝트"** 클릭

### 4.2 필터 사용

| 필터 | 설명 |
|------|------|
| **고객사** | 특정 고객사 프로젝트만 표시 |
| **소속** | 전장/유압/기타 부문별 표시 |
| **상태** | 진행중/완료/보류 상태별 표시 |
| **즐겨찾기** | 즐겨찾기 프로젝트만 표시 |

### 4.3 프로젝트 상세

프로젝트 클릭 시 확인 가능한 정보:
- 기본 정보 (고객사, ITEM, 담당자 등)
- 단계 진행 현황
- 간트 차트
- 업무일지 이력
- 이슈 현황

## 5. 업무일지 조회

### 5.1 목록 조회

사이드바에서 **"📝 업무일지"** 클릭

### 5.2 필터 사용

| 필터 | 예시 |
|------|------|
| **날짜 범위** | 이번 주, 이번 달, 특정 기간 |
| **프로젝트** | 특정 프로젝트의 업무일지만 |
| **담당자** | 특정 담당자의 업무일지만 |

### 5.3 이슈 현황 파악

- 이슈 있는 업무일지는 빨간색으로 표시
- 미해결 이슈 우선 확인

---

## 6. 고급 검색

### 6.1 복합 조건 검색

여러 조건을 조합하여 업무일지를 검색할 수 있습니다:

| 조건 조합 | 용도 |
|-----------|------|
| 기간 + 고객사 | 이번 달 A사 업무 현황 파악 |
| 담당자 + 이슈 | 특정 담당자의 이슈 사항 확인 |
| 단계 + 키워드 | 개발 단계에서 "테스트" 관련 업무 |

### 6.2 자주 사용하는 검색 저장

1. 검색 조건 설정
2. **[⭐ 저장]** 클릭
3. 이름 입력 (예: "이번 주 A사 현황")
4. 다음에 **[📂 저장된 검색]** 드롭다운에서 바로 불러오기

## 7. 주간보고

### 7.1 조회 방법

1. 사이드바에서 **"📋 주간보고"** 클릭
2. 주차 선택하여 해당 주 보고서 확인

### 7.2 보고서 구성

| 섹션 | 내용 |
|------|------|
| **보고 항목** | 구분별(농기/중공업/해외/기타) 프로젝트 진행 현황 |
| **공지사항** | 요청사항 및 공지 |
| **주간 보고 요약** | 금주 주요 진행사항 및 차주 계획 |

## 9. 활용 팁

### 9.1 일일 체크 루틴

1. **경영진 대시보드** 확인
   - 지연 프로젝트 유무 확인
   - 주요 프로젝트 카드 상태 확인

2. **이슈 있는 업무일지** 확인
   - 고급 검색에서 이슈 필터링
   - 미해결 이슈 팔로업

### 9.2 주간 체크 루틴

1. **주간보고** 검토
   - 각 프로젝트 진행 상황 파악
   - 차주 계획 확인

2. **계획vs실적** 분석
   - 지연 프로젝트 원인 파악
   - 필요시 자원 재배치 검토

### 9.3 월간 체크 루틴

1. **전체 일정 (간트차트)** 확인
   - 향후 일정 파악
   - 병목 구간 확인

2. **회의록** 검토
   - 주요 의사결정 사항 확인
   - 후속 조치 확인

## 10. 자주 묻는 질문

### Q1. 프로젝트를 직접 수정할 수 있나요?

> 경영진 역할은 조회 중심입니다.
> 수정이 필요하면 담당자 또는 관리자에게 요청해주세요.

### Q2. 즐겨찾기는 몇 개까지 등록할 수 있나요?

> 제한 없이 등록 가능합니다.
> 단, 대시보드에서 한 눈에 보기 위해 5~10개 정도를 권장합니다.

### Q3. 특정 담당자의 업무만 보고 싶습니다.

> 고급 검색에서 담당자 필터를 사용하세요.
> 자주 사용하면 검색 조건을 저장해두세요.

### Q4. 코멘트는 누가 볼 수 있나요?

> 프로젝트 코멘트는 해당 프로젝트에 접근 권한이 있는 모든 사용자가 볼 수 있습니다.
> 프로젝트 팀원, 관리자, 경영진 모두 확인 가능합니다.

### Q5. PDF 다운로드는 어떻게 하나요?

> **계획vs실적 비교** 화면에서 **[📥 PDF]** 버튼을 클릭하면
> 현재 화면을 PDF로 다운로드할 수 있습니다.
